1. netkraken

netkraken | network analyzer

tl;dr

  • records network connections of a bunch of hosts, continuously
  • aggregates connection data: minutes, hours, days
  • analyzes and displays connection data

Features

  • connection recorder (aka minion) compatible with python2.6/2.7/3.3/3.4/pypy
  • aggregation runs on minion, scales well
  • communication over ssh
  • multi-dimensional analyzes, configurable via url
